- ZF1 Breaking change

- Les différents types de configs dans ZF2
    - system config      => config/application.config.php
    - module config      => module.config.php
    - application config => autoload/*

- Config system
    - Liste/activation des modules
    - Configuration cache de configuration modules/application
    - Configuration ordre de chargement des fichiers de configuration modules/application
    
- Config modules
    - Ordre de chargement FIFO
    - Configuration liée au module *uniquement*
    - Surchargé par la configuration d'application

- Config application
    - Ordre de chargement GLOB (défini dans la config système)
        global.php
        *.global.php
        APPLICATION_ENV.php
        *.APPLICATION_ENV.php
        local.php
        *.local.php    

    - Surcharge les configurations des modules

- Opérations de merging sur les fichiers de config
  - Ordre de merging (modules, application)
  - glob
  - \Zend\Stdlib\ArrayUtils::merge (différence avec array_merge_recursive)
  - \Zend\Stdlib\Glob::glob
